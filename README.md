# Word Clock

![build status](https://travis-ci.org/fdomig/clock.svg?branch=master)

A clock returning time as a string written in Kotlin and tested with JUnit 5. E.g. `07:43:12` will be returned as `It is quarter to eight`.

## Build & Test

```bash
$ mvn clean test
```

## Run

```bash
$ mvn exec:java
```
